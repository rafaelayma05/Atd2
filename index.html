Meus Projetos

Projeto 1: Site de Loja de Roupas

Este projeto consiste em um site de e-commerce para uma loja de roupas. O site foi desenvolvido utilizando HTML e CSS, focando na exibição dos produtos de maneira atraente e organizada. Os usuários podem navegar pelas categorias, visualizar os detalhes das peças e realizar compras online. O design responsivo garante que a loja seja acessível em dispositivos móveis e desktops, oferecendo uma experiência agradável aos clientes que buscam comprar roupas de forma prática e rápida.

Projeto 2: Sistema de Vendas para Loja de Carros

Este projeto é um sistema de vendas desenvolvido para uma loja de carros. O objetivo é facilitar a gestão do estoque de veículos, permitindo que os usuários vejam os carros disponíveis, suas características, preços e realizem consultas de financiamento. Com uma interface desenvolvida em HTML e CSS, o sistema é otimizado para uma experiência fluida em diferentes dispositivos, garantindo que os clientes possam acessar as informações de forma simples e rápida.
